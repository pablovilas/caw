# Messaging

## Preferred Tagline

Watch every coding agent from one quiet perch.

## Alternate Taglines

- The ambient control layer for coding assistants.
- Quiet observability for AI coding sessions.
- Track active coding agents without losing flow.

## Short Product Description

`caw` is a desktop and terminal watcher for active AI coding sessions across tools, projects, and apps.

## One-Sentence Pitch

`caw` gives you a single, low-noise view of every coding assistant that is currently working, waiting, idle, or gone.

## Slightly Longer Pitch

`caw` watches the coding assistants already running on your machine and turns them into one calm operational view. You can scan status, group sessions, and jump back to the right terminal without digging around.

## Microcopy Style

- Prefer verbs that imply control without drama.
- Keep labels short and technical.
- Use lower-case for status names in product surfaces.
- Avoid exclamation marks.

## Good UI Phrases

- `No active sessions`
- `Waiting for input`
- `Focus terminal`
- `Group by project`
- `Last message`
- `Status`

## Phrases To Avoid

- `Your AI team`
- `Boost productivity`
- `Never miss a beat`
- `Magic`
- `Delightful`
